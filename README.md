# PaperilyGPT-论文一键免费解析助手

## 项目简介

PaperilyGPT是一个Chrome浏览器插件，专为科研人员设计，提供论文一键解析、搜索和管理功能，让科研更高效。

## 功能特性

- **一键论文解析**：自动识别PDF论文页面，支持学术网站的PDF链接自动识别和转换，点击解析按钮即可获取论文摘要和关键信息
- **多平台论文搜索**：集成谷歌学术、百度学术和共享资源的搜索和下载
- **用户系统**：支持登录/注册，保存个人阅读历史和收藏

## 使用说明

### 解析论文

1. 打开任意PDF论文页面（支持直接PDF链接、IEEE Xplore和ACM等学术网站）
2. 页面会自动显示"解析"按钮
3. 点击"解析"按钮，插件会自动下载PDF并上传到服务器进行分析
4. 分析完成后，会显示论文摘要和关键信息

### 搜索论文

1. 点击插件面板上的"搜索"按钮
2. 在搜索框中输入论文标题
3. 点击"搜索"按钮，插件会显示谷歌学术、百度学术和共享资源的搜索结果
4. 点击相应链接查看详细信息或者直接下载PDF文件

### 登录/注册

1. 点击插件面板上的"登录"按钮
2. 在弹出的登录界面中输入邮箱和密码
3. 点击"登录"按钮
4. 如需注册新账号，点击"没有账号？去注册"链接

### 收藏论文

1. 登录后，解析论文页面会显示"收藏"按钮
2. 点击"收藏"按钮即可将论文添加到收藏列表
3. 再次点击"取消收藏"按钮可移除收藏

## 注意事项

- 插件需要网络连接才能分析论文
- 解析大型PDF文件可能需要较长时间
- 部分学术网站可能需要登录才能访问PDF文件
- 插件会使用IndexedDB存储分析结果和阅读历史
